# Windows Server Active Directory Lab

## Overview

This project is a home lab created to practice system administration and IT support skills.

The goal was to simulate a small business environment using Windows Server and client machines.

## Environment

- Windows Server 2022
- Windows 11 Client
- Windows 10 Client
- Hyper-V / Virtualization

## Completed Tasks

### Active Directory
- Installed Active Directory Domain Services (AD DS)
- Created a domain controller
- Created users and groups
- Managed user permissions

### DNS & DHCP
- Configured DNS services
- Practiced DHCP scope configuration
- Tested client network communication

### Group Policy
- Created and tested Group Policies
- Managed user and computer settings

### Client Management
- Joined Windows client computers to the domain
- Tested login and permissions

## Skills Practiced

- Windows Server Administration
- Active Directory
- DNS
- DHCP
- User Management
- Troubleshooting
- Network Basics

## Tools Used

- Windows Server 2022
- Windows 11
- Windows 10 
- Hyper-V
